# Amera Eid — Website Redesign & CRO Audit

A conversion-focused redesign of [ameraeid.com](https://ameraeid.com), the home of Egyptian belly dance artist, teacher and mentor **Amera Eid**.

## Live preview

Once GitHub Pages finishes building, the site is served from the repository root:

- **Home** — `/index.html`
- **About** — `/about.html`
- **Private Tuition** — `/classes.html`
- **Workshops** — `/workshops.html`
- **Tours to Lebanon** — `/tours.html`
- **Mentoring** — `/mentoring.html`
- **Contact** — `/contact.html`

## What's inside

| Path | Description |
|------|-------------|
| `*.html`, `styles.css` | The 7-page static redesign. Aesthetic: "Levantine editorial" — Cormorant Garamond + Jost, aubergine / gold / parchment. |
| `audit/AmeraEid-CRO-Audit.pdf` | Full CRO audit report (scorecard, page-by-page findings, prioritized roadmap). |
| `audit/cro-report.html` | HTML source of the report. |

## CRO highlights addressed

The original site is an elegant brochure with no booking path — every service funnels to one generic contact form. This redesign adds:

- A real hero headline + primary call-to-action and a 40-year proof bar.
- Four bookable offer cards routing to each service.
- Priced, bookable packages for tuition and mentoring.
- A dated, reservable workshop schedule plus a categorized topic library.
- A complete Tours to Lebanon landing page with itinerary, inclusions, gallery and a deposit-to-reserve flow.
- Testimonials, FAQs and warmer contact forms.

> **Note:** Prices and testimonials are placeholders for layout and must be replaced with Amera's real rates and reviews before launch. The "Book" buttons are prototype links — wire them to Shopify checkout / booking products to take payment.
